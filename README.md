# MyHotelBooking
A Hotel Room Booking Website (Acadamic Work)
<br/>
Developed with PHP, HTML/CSS and MySQL
<hr/>
<br/>
Make sure to have install XAMPP 
1. Stop the MySQL in Services on the device and open XAMPP to start Apache and MySQL 
2. Click the Admin button on MySQL in XAMPP to access phpMyAdmin
3. Import the "myhotelbooking.sql" into phpMyAdmin to get the myhotelbooking database
4. Click on explorer in XAMPP and insert the "myhotelbooking folder" into "htdocs folder" in the xampp
Then, you are all set to use the website.

The link to access the website: http://localhost/myhotelbooking/index.php
You can create a new account or use the existing ones which you can find in the user table in the myhotelbooking database on phpMyAdmin

The link to access the admin panel of the website: http://localhost/myhotelbooking/admin/
The login for the admin panel(Admin Name: hoteladmin, Password: 12345)
This also can be found in the admin_cred table in the myhotelbooking database on phpMyAdmin

Ensure that there are no other "myhotelbooking" databases in phpMyAdmin and a "myhotelbooking" folder in htdocs folder, except for this one.
